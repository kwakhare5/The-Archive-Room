## 🧐 What is The Archive Room?

The Archive Room is a professional-grade "Memory Palace" for AI agents. It transforms complex backend operations—such as **RAG (Retrieval-Augmented Generation)**, **Reasoning Steps**, and **Tool Usage**—into a high-fidelity, pixel-art simulation. 

Watch your agents physically interact with your data nexus:
- 📚 **Bookshelf**: Represents Vector Database retrieval (RAG).
- 🧠 **Whiteboard**: Represents reasoning/thinking phases.
- 🗑️ **Bin**: Represents technical debt cleanup or session resets.

## 🚀 Getting Started

### Prerequisites
- **Node.js 20+**
- **Python 3.11+**
- **Supabase Account**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kwakhare5/The-Archive-Room.git
   cd The-Archive-Room
   ```

2. **Frontend Setup**
   ```bash
   npm install
   npm run dev
   ```

3. **Backend Setup**
   ```bash
   cd backend
   pip install -r requirements.txt
   python main.py
   ```
